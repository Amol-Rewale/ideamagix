# IDEAMAGIX

A responsive personal portfolio website built using **HTML, CSS, and JavaScript**. The design is modern, sleek, and fully responsive for both desktop and mobile devices.

[Live Demo](https://6971da259050ca85b453f27a--bright-entremet-61dd51.netlify.app/)

---

## Features

- **Responsive Design**: Optimized for both desktop and mobile devices.
- **Lazy Loading Images**: Improves performance by loading images only when they appear in the viewport.
- **Owl Carousel Sliders**: Smooth, interactive sliders to showcase projects and content.

---

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **Libraries/Plugins**: Owl Carousel

---

## Project Sections

1. **Home**
   - Introduction with name, profession, and a featured image.

2. **About Me**
   - Shows experience, completed projects, and expertise.
   - Sections for coding, infrastructure, and web development skills.

3. **Projects**
   - Interactive project cards with descriptions and links.
   - Owl Carousel slider for showcasing multiple projects.

4. **Contacts**
   - Contact form with name, email, phone, and message fields.
   - Social media links.
